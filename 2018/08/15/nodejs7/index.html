nodejs7 express中cookie使用

  • nodejs


**

  • 使用cookie-parser
  • 1 npm install cookie-parser
  • 2 引入模块 var cookie=require(“cookie-parser”);
  • 3 注册cookie中间件 app.use(cookie())
  • 4 设置cookie res.cookie(name,value,options:CookieOptions)
  • 其中 name:cookie名 value cookie值 options cookie其他配置项,此参数可选
  • export interface CookieOptions {
    maxAge?: number;//最大失效时间(毫秒),设置在多久后失效
    signed?: boolean;//表示是否签名 cookie, 设为 true 会对这个 cookie 签名，这样就需要用
    res.signedCookies 而不是 res.cookies 访问它。被篡改的签名 cookie 会被服务器拒绝，并且 cookie
值会重置为它的原始值
    expires?: Date | boolean;//过期时间
    httpOnly?: boolean;//是微软对 COOKIE 做的扩展。如果在 COOKIE 中设置了“httpOnly”属性，则通过程序（JS
    脚本、 applet 等）将无法读取到 COOKIE 信息，防止 XSS 攻击产生
    path?: string;//cookie影响的路径 如果路由不匹配则不会发送cookie
    domain?: string;//域名
    secure?: boolean | ‘auto’;//当此值为true,表示cookie在http中无效,在https中有效
    encode?: (val: string) => void;
    sameSite?: boolean | string;
    }
  • 获取cookie的值 req.cookies.name
  • 删除cookie
  • res.cookie(‘rememberme’, ‘’, { expires: new Date(0)});
    res.cookie(‘username’,’zhangsan’,{domain:’.ccc.com’,maxAge:0,httpOnly:true});
  • */
